# 🎉 Happy Birthday Animation

## Preview

![Birthday Animation](screenshot.png)

A Python birthday celebration project built using Pygame.

## Features

* 🎂 Birthday Cake
* 🎈 Floating Balloons
* 🎆 Fireworks Animation
* 🎊 Confetti Effects
* 🎵 Background Birthday Music
* ✨ Animated Birthday Message

## Requirements

* Python 3.x
* Pygame

## Installation

```bash
pip install pygame
```

## Run the Project

```bash
python birthday.py
```
